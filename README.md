# **Password Generator**

A web application that generates passwords. The user can set the range of characters that may appear in the password.

**Options:**
- Password length
- Uppercase letters
- Numbers
- Lowercase letters
- Symbols
- Polish characters


The minimum password length is 7, and the maximum is 50.
